Installiation

  • firstly you can download this codes github
  • open 2 terminal
  • first terminal has be back end codes
  • second terminal has be front end codes

for back end

  • open back end folder and write this codes 

    •  >> npm install -g
  • and open app folder after write this codes 

    •  >> node server.js

for front end

  • open front end folder write this codes 

    •  >> npm install -g
  • and open src folder after write this codes 

    •  >> ng serve

for database (mysql)

  • open mysql front or what do you use 

    •  create database your_db_name;
  • after change this code in Back_end > server > app > config.js 

    •  name: 'your_db_name'

---> http://localhost:4200

---> Enjoy the codes ...
